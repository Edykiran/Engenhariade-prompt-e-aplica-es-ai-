## Sobre a Disciplina
Este repositório é dedicado à disciplina **Engenharia de Prompt e Aplicações em Inteligência Artificial**.

### Objetivo
Explorar técnicas e metodologias para projetar, otimizar e avaliar interações com modelos de linguagem e sistemas de IA generativa.

### Foco Principal
Transformar ideias em instruções eficazes que resultem em respostas:

- **Úteis**
- **Criativas**
- **Confiáveis**
